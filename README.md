# Personal Introduction Program

## Project Overview
This project is a simple Python program that interacts with users by asking for their personal information such as name, age, hobby, and city. The program then generates a friendly welcome message using the collected information.

The goal of this project is to understand basic Python concepts such as user input, variables, and formatted output.

---

## Objectives
- Learn how to use the `input()` function
- Store user data using variables
- Display personalized messages using `print()`
- Understand basic Python program structure
